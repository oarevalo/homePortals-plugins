This module displays pictures from Flickr filtered by tags
or by username.

Installation:
- Unpack module files into the modules directory
- Add your Flickr API key to the module-properties.xml file
    <module name="Home.Modules.flickrFeed.flickrFeed">
        <property name="API_Key" value="--- your API KEY here ---" />
    </module>
  * You may obtain an API key from the flickr website
- Restart HomePortals
