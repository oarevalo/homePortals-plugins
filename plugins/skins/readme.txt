Instructions:

1. Add the following to the <plugins> section of your application's homePortals-config.xml.cfm

<plugin name="skins" />

2. To set a skin for a page, set a page property named "skinID" to the ID of the resource of 
type "Skin" that you want to apply to the page. You can set the "skinID" page property on the app's 
config file so that it is applied by default to all pages.
